Steps from hypr wiki, each of this may have requirements, install them first
Install hyprland
    $ sudo pacman -S hyprland
    Change default terminal from kitty to ghostty
    $ nvim ~/.config/hypr/hyprland.conf
    Search for "$terminal"
    Set up "$terminal = ghostty"
    Get ride of the warning by eliminating the line with:
    autogenerated=1
Monitor
    https://wiki.hypr.land/Configuring/Monitors/
    $ hyprctl monitors
    Put the output of that or my example
    $ nvim ~/.config/hypr/hyprland.conf
    Change default monitor to
    monitor=,1920x1080,auto,1
Systemd startup
    $ sudo pacman -S --needed uwsm libnewt
    $ cp forArch/.zprofile .
XDG Desktop Portal
    $ sudo pacman -S --needed xdg-desktop-portal-hyprland
Audio server
    $ sudo pacman -S --needed pipewire wireplumber
    $ flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
    $ flatpak install flathub com.saivert.pwvucontrol
Authentication daemon
    $ sudo pacman -S hyprpolkitagent
Qt wayand support
    $ sudo pacman -S --needed qt5-wayland qt6-wayland
App launcher
    $ sudo pacman -S --needed rofi
    $ cp -r forArch/.config/rofi/ .config/
    $ cp forArch/rofi/themes/t4-s4.rasi /usr/share/rofi/themes/
    $ cp forArch/rofi/themes/tokyoNight.rasi /usr/share/rofi/themes/
    To change the theme modify .config/rofi/config.rasi
Clipboard manager
    Check that installs wl-clipboard
    $ yay -S --needed clipse
Status bar
    $ sudo pacman -S --needed waybar
    Do i need this font??
    $ sudo pacman -S ttf-font-awesome
Auto mounting for usb and external devices
    $ sudo pacman -S --needed udiskie
Notification daemon
    $ sudo pacman -S dunst
    Others: mako, swaync, fnott

---------

After everything works, check the following.
- Menu logout: wlogout
- Color temeperature/gamma change: hyprsunset
- idle management: hypridle
- screen lock: hprlock
- system info: hyprsysteminfo
- cursor theme: hyprcursor
- screenshot: $ yay -S hyprshot
