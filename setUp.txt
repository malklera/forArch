    All commands are run from home
    Upgrade
        $ pacman -Syu
    Microcode
        $ sudo pacman -S amd-ucode
    Network manager
        $ sudo pacman -S networkmanager
        Do i need to do this?
        $ systenctl enable NetworkManager.service

    Better man page:
        $ sudo pacman -S tldr
    Font
        $ sudo pacman -S ttf-jetbrains-mono-nerd
        $ sudo pacman -S ttf-nerd-fonts-symbols
        $ sudo pacman -S ttf-awesome
    Set up SSH
        $ sudo pacman -S openssh       
        Generate key pair commenting who and when it was created
        $ ssh-keygen -C "$(whoami)@$(uname -n)-$(date -I)"
        Default file location to save key
        Write the passphrase on a piece of paper, not on the pc, 64 characters
        easy to remenber
    Git
        $ sudo pacman -S --needed git
        $ git config --global user.name "malklera"
        $ git config --global user.email "malklerulo@gmail.com"
        $ git config --global init.defaultBranch main
        Get ssh passphrase to clone my repositories
    Get backup configs
        Try cloning using ssh and see if the key is used automatically
        $ git clone git@github.com:malklera/forArch.git
        If that do not work, use this
        $ git clone https://github.com/malklera/forArch
        Or better, use my ssh key so i do not have to put my credentials for each push
    Shell
        $ sudo pacman -S zsh
        Initial config
        $ zsh
        Get the path to zsh to use below
        $ chsh -l
        To set zsh as the new default
        $ chsh -s /full/path/to/shell
        Copy from backup
        $ cp forArch/.zshenv .
        $ cp -r forArch/.config/zsh/ .config
    Terminal
        $ pacman -S ghostty
        $ cp -r forArch/.config/ghostty .config
    AUR helper
        $ sudo pacman -Syu
        $ sudo pacman -S --needed base-devel
        $ git clone https://aur.archlinux.org/yay.git
        $ makepkg -si
        The following commands come from here
        https://github.com/Jguer/yay

        $ yay -Syu --devel
        $ yay -Y --devel --save
        $ sudo rm -r yay
